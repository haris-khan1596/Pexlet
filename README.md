# Pexlet

A modern web application showcasing technology solutions for restaurant businesses. Pexlet specializes in streamlining restaurant operations through innovative software solutions.

## About

Pexlet provides advanced technology solutions designed to improve the sales and operations of restaurants. The platform offers expertly designed software to streamline processes, enhance customer engagement, and increase revenue.

## Services

- **POS System** - Streamlines restaurant operations, enhances business performance, and increases customer satisfaction
- **Restaurant Management System** - Comprehensive management system that empowers owners to manage efficiently and delight customers
- **Website Design** - Custom, visually stunning websites that showcase the unique ambiance and cuisine of restaurants

## Tech Stack

- **Backend**: Node.js with Express.js
- **Template Engine**: EJS with EJS-Mate
- **Frontend**: HTML, CSS, JavaScript
- **Styling**: Bootstrap (responsive design)

## Project Structure

```
Pexlet/
├── app.js                 # Main application entry point
├── package.json           # Project dependencies and scripts
├── public/                # Static assets
│   ├── images/           # Image files
│   ├── stylesheets/      # CSS files
│   ├── javascripts/      # Client-side JavaScript
│   ├── fonts/            # Custom fonts
│   └── video/            # Video content
└── views/                # EJS templates
    ├── layouts/          # Layout templates
    ├── partials/         # Reusable components (navbar, footer)
    └── ejs/              # Page sections (banner, about, services, etc.)
```

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Pexlet
```

2. Install dependencies:
```bash
npm install
```

## Running the Application

Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:3000`

## Dependencies

- **express** (^4.18.2) - Web application framework
- **ejs** (^3.1.9) - Template engine
- **ejs-mate** (^4.0.0) - Layout and partial support for EJS
